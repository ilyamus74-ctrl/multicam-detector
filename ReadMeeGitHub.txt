Вариант 1 (обычный, без переписывания истории)

Из корня проекта (~/rknpu2/examples/multicam_yolovo_v1):

git status                         # посмотреть, что изменено/неотслеживается
git add -A                         # добавить абсолютно всё (включая новые и удаления)
git commit -m "sync: текущее рабочее состояние с платы"
git push                           # отправить в origin/main

Этого обычно достаточно. main.cc и main.cc.b3 уйдут в репозиторий.
